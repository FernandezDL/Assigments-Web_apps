# Assigments - JavaScript Web Apps
### Vancouver Film School

## Description
_Crazy Mad Flying Animals_ is a physics-based 2D web game in which the player tries to win all the levels by throwing Crazy Mad Flying Animals to defeat all the enemies.

## Assigment 1
The goal of this assigment is to creat a client-server aookucatuib that allows the editing of levels for the game, allowing to create blocks, save, and load levels

In the editor the level designer can place 5 different block types:
- Block
- Enemies (Pigs)
- Catapult
- Dirt
- Rocks
  
## Assigment 2
The second assigment goal is to use the saved levels and create the game.
